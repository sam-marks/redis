# Run Redis in Docker Desktop
Install Docker Desktop:
https://www.docker.com/products/docker-desktop/

Install Python:
https://www.python.org/downloads/

Run:
```
 docker-compose up -d
 ```

 Run: 
 ```
 pip install redis
 ```

Run: 
```
pip install colorama

```

Run:
```
python.exe -m pip install --upgrade pip
```

Run:
```
python .\test_redis.py
```
